# 📊 AJOL Journal Analytics Dashboard

### Publication Trends, Open Access Distribution, and Journal Performance Analysis (2020 – March 2024)

## 📌 Project Overview

This project analyzes publication data from **African Journals Online (AJOL)** to uncover trends in scholarly publishing across Africa. The data was cleaned and transformed using **Microsoft Excel** and **Power Query**, followed by data modelling and interactive dashboard development in **Power BI**.

The dashboard provides insights into publication trends, journal performance, Open Access distribution, geographical coverage, and journal quality based on the **Journal Publishing Practices and Standards (JPPS)** classification.

---

## 🎯 Objectives

- Analyze publication trends between 2020 and March 2024.
- Identify journals with the highest publication output.
- Examine Diamond and Non-Diamond Open Access distribution.
- Analyze country-wise publication performance.
- Evaluate journal quality using JPPS classification.
- Develop an interactive Power BI dashboard for research analytics.

---

## 📈 Key Insights

- Analyzed **50,000+ research articles** from **739 journals** across **39 countries**.
- Publication output increased steadily from **2020 to 2023**.
- **Pan African Medical Journal** recorded the highest number of published articles.
- **65.49%** of journals follow the **Diamond Open Access** publishing model.
- **Nigeria, South Africa, and Uganda** are the leading contributors to research publications.
- Most journals are classified as **1 Star** under the JPPS quality framework.

---

## 💡 Business Recommendations

- Improve journal quality to achieve higher JPPS ratings.
- Encourage research participation from lower-contributing countries.
- Continue promoting the Diamond Open Access publishing model.
- Support collaboration between established and emerging journals.
